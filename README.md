# Lista de Tareas

Aplicación sencilla de lista de tareas desarrollada con HTML, CSS y JavaScript.  
Permite agregar tareas, marcarlas como completadas y eliminarlas cuando ya no sean necesarias.

## Enlace al proyecto
https://erbusaa06.github.io/lista-de-tareas/

## Organización del proyecto
- index.html → Página principal
- styles.css → Estilos de la aplicación
- script.js → Lógica de funcionamiento
- README.md → Información del proyecto

## Funcionalidades
- Agregar nuevas tareas
- Marcar tareas como completadas
- Eliminar tareas completadas
- Mensaje cuando no hay tareas
- Interfaz moderna y responsive

## Cómo usar
1. Escribe una tarea en el campo de texto.
2. Pulsa el botón "Agregar" o presiona Enter.
3. Marca una tarea como completada con el botón correspondiente.
4. Elimina tareas completadas cuando ya no las necesites.
